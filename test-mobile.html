✅ Test Navbar Mobile

📱 Instruksi Test:

  1. Buka halaman ini di HP Android Anda
  2. Klik tombol hamburger (≡) di kanan atas
  3. Menu drawer harus muncul dari kanan
  4. Klik tombol × di kanan atas drawer untuk menutup
  5. Atau klik area gelap di luar drawer

Status Navbar:

Mobile (< 900px): Hamburger terlihat, nav links tersembunyi

Desktop (≥ 900px): Nav links terlihat horizontal, hamburger tersembunyi

Fitur yang Harus Bekerja:

  • ✅ Klik hamburger → drawer slide dari kanan
  • ✅ Klik tombol × → drawer tutup
  • ✅ Klik overlay hitam → drawer tutup
  • ✅ Klik link menu → drawer tutup
  • ✅ Navbar tidak terpotong di bagian atas
  • ✅ Navbar full-width di mobile

⚠️ Jika navbar masih bermasalah:
1. Clear cache browser (Settings → Privacy → Clear browsing data)
2. Atau buka dalam mode Incognito/Private
3. Hard refresh: Ctrl+F5 (PC) atau pull to refresh (Mobile)
